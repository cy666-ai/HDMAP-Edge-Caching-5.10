步骤 1 — 生成输入文件（可选，后端启动后会自动生成）：

cd "simmap1.0/backend"
node scripts/generateMatlabInput.js

步骤 2 — 运行 Python 算法生成缓存决策：
cd "algorithm"
python hm_export_cache_decision.py

步骤 3 — 启动后端和前端（与之前一样）：
cd "simmap1.0/backend"
npm run dev
cd "simmap1.0/frontend"
npm run dev

步骤 4 — 打开浏览器，点击"开始行驶"，即可在地图上看到RSU 标记点和实时的命中率面板。

注意：后端会在每 5 个时间片自动触发 Python 算法重算，无需手动执行步骤 2。
